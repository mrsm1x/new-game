# Neon Player Hub

Мобильный веб-интерфейс игровой платформы с 4 разделами и нижней навигацией. Чисто фронтенд — данные хранятся в `localStorage`, готово к деплою на GitHub Pages.

## Возможности

- **Профиль** — аватар с рамкой топа, баланс, пополнение, рефералы (+1% к базовым 4% за 3 / 10 / 25 рефералов), реферальная ссылка
- **Инвестиции** — депозит раз в 24 часа, прибыль = сумма × ваш %
- **Игры** — Краш, Мины, Монетка (ставки с основного баланса) + live-лента ботов
- **Топ** — рейтинг по монетам, золото / серебро / бронза рамки для топ-3 (отображаются в профиле)

## Запуск

```bash
npm install
npm run dev
```

## Сборка для GitHub Pages

```bash
npm run build
```

Содержимое папки `dist/` загрузите в репозиторий. Для GitHub Pages добавьте в `vite.config.ts`:

```ts
export default defineConfig({
  base: '/имя-репозитория/',
  // ...
})
```

## Стек

- React 19 + TypeScript + Vite
- Zustand (с persist в localStorage)
- Framer Motion (анимации)

## Структура

```
src/
├── components/
│   ├── profile/       # Профиль и пополнение
│   ├── investment/    # Инвестиции
│   ├── games/         # Краш, Мины, Монетка, боты
│   ├── leaderboard/   # Топ игроков
│   ├── layout/        # Нижняя навигация
│   └── shared/        # UI-компоненты
├── store/             # Zustand store
├── utils/             # Рефералы, боты, форматирование
└── types/
```

## Демо-кнопки

- «Симулировать реферала» на странице профиля — для теста роста %
- Баланс и прогресс сохраняются между перезагрузками
